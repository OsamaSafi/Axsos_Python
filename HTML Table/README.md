# Flask User Table App

A simple web application built using Flask and Bootstrap 5 that displays user information inside a responsive table using Jinja templating.

---

## 📌 Features

- Built with Flask
- Responsive Bootstrap 5 design
- Dynamic table rendering using Jinja2
